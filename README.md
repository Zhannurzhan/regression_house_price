# Housing Regression MLOps

End-to-end machine learning pipeline for housing price prediction with XGBoost,
MLflow experiment tracking, FastAPI model serving, Streamlit dashboard, Docker
containers, and GitHub Actions deployment scaffolding.

## Project Structure

```text
src/feature_pipeline/      Data loading, preprocessing, feature engineering
src/training_pipeline/     Training, tuning, evaluation
src/inference_pipeline/    Production inference logic
src/api/                   FastAPI service
src/batch/                 Batch prediction jobs
tests/                     Unit and integration tests
notebooks/                 EDA and experimentation notebooks
```

## Local Setup

```bash
uv sync
```

Run tests:

```bash
uv run pytest
```

Run the API:

```bash
uv run uvicorn src.api.main:app --host 0.0.0.0 --port 8000
```

Run the dashboard:

```bash
uv run streamlit run app.py --server.port 8501
```

## MLOps Status

- Local data and feature pipelines are implemented.
- Training, tuning, inference, and batch prediction modules are implemented.
- FastAPI and Streamlit services are scaffolded.
- Dockerfiles exist for API and dashboard containers.
- GitHub Actions CI/CD workflow is scaffolded for AWS ECR/ECS deployment.

Large local artifacts such as `data/`, `models/`, `mlruns/`, and `mlflow.db` are
ignored by Git and should be stored in S3 or regenerated by the pipeline.
